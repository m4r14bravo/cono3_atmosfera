Contenidos del tema

La atmósfera: qué es, para qué sirve (filtra, regula, contiene) y sus capas (troposfera, estratosfera, capas exteriores)
Fenómenos atmosféricos: nubes, precipitaciones, viento, tormentas
El ciclo del agua: evaporación, formación de nubes, lluvia, vuelta al mar
Instrumentos: termómetro, anemómetro, veleta, pluviómetro
Mapas del tiempo y meteorología
Las cuatro estaciones: tiempo y paisaje de cada una
Contaminación del aire: causas y formas de cuidarlo
